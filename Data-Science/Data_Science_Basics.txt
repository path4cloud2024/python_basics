## Basic Descriptive Statistics
1) minimum: the smallest value in a collection of values
2) maximum: the largest value in a collection of values
3) range: difference between the maximum and minimum
4) count: number of values in a collection
5) sum: total of the values in a collection

--> Measure of Dispersion (also called Measures of Variability)
    - range
    - help determine how spread out the values are
    - variance and standard deviation

--> Measures of Central Tendency - Mean, Median, and Mode
        Central Tendency: is the central value of your set or sequence of values.
        Mean: average value in a set of values.
        Median: the middle value when all the values are arranged in sorted or ascending order.
        Mode: most frequently occurring value.
